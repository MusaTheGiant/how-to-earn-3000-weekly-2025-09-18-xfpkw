⛏GETTING STARTED

Getting Started with TexitCoin

Follow these simple steps to join TexitCoin and start receiving daily TXC mining rewards for life. Complete setup takes less than 10 minutes.

1

Create Exchange Account

Set up your exchange account to receive daily TXC rewards. Choose one option:

Create BitMart Account Or Dex-Trade (if BitMart unavailable)
2

Prepare USDC Payment

Ensure you have USDC ready in your exchange account before starting registration.

Important: Add extra $10 for transaction fees per package

Payment must be completed within 60 minutes of starting registration.

3

Complete Registration

  • Choose your Affiliate ID/username
  • Select "Crypto" as payment method
  • Verify sponsor ID shows: mtglegacy
  • Add your TXC wallet address for rewards
  • Leave "Cold Storage Coin" blank
Start Registration Now
4

Secure Payment

CRITICAL: Always select Base chain when paying with USDC

Wrong network selection can result in permanent fund loss. Double-check before confirming payment.

Stay on payment page until you see confirmation message.

5

Dashboard Setup

  • Check email for MineTXC welcome message
  • Create login password as instructed
  • Complete profile in "My Account" page
  • Set commission preferences (USDC/HASH/TXC)
  • Verify TXC wallet address for daily rewards

Your daily TXC mining rewards will begin once setup is complete!
